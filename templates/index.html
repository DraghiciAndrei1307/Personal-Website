PostgreSQL Ansible Automation

The story behind this project is as follows. At work, we were using Ansible playbooks and Python packages so that we can automate the day-to-day tasks of a PostgreSQL DBA. At that time, I did not have the experience with any of the two technologies. So, I started to follow Udemy courses in order to learn what was needed. This project is basically my struggle in applying the knowledge about PostgreSQL, Ansible and Python in a real project, project that tries to have similar functionality as the one we are using at work, but at a much smaller scale.
